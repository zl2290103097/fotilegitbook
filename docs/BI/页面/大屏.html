1. 大屏指标计数规则

a.销售额：订单台量 * 均价 ( DRP门店渠道为KA（含传统百货）的取“单价”，其他渠道取“信用价”)

b.订单台量：DRP录入订单数量（多台）

c.线索数：云平台线索录入数量

d.平均店效：销售额 / DRP非虚拟门店

e.成交顾客数量：DRP录入订单个数（2C用户数)

f.客单价：销售额 / 成交顾客数量

g.客单台：订单台量 / 成交顾客数量
